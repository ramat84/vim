# Normal Mode
* {G}, {gg}, {^}, {$} - Start/End
* {f}, {t} - Jump to character 
* {%}, {✱}, {⦄} - Jump
* {w}, {b}, {e}, {t} - Words, Tag
* {c}, {d} - Change, Delete 
* {a}, {i} - Outside, Inside
